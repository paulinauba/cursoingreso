# Curso de Ingreso - Aplicación Web

Esta aplicación fue desarrollada para acompañar el curso de ingreso del Colegio UBA Escobar.  
Está construida con **Node.js**, **Vite** y **Firebase**.

## 🚀 Funcionalidades principales
- Registro y autenticación de estudiantes.
- Acceso a materiales y recursos del curso.
- Panel de administración para el área de sistemas.
- Integración con Firebase para base de datos y hosting.

## 📂 Estructura del proyecto
- `/src` → Código fuente de la aplicación.
- `/public` → Archivos estáticos.
- `/dist` → Carpeta de build generada por Vite (no se sube al repo).
- `.gitignore` → Configuración para excluir archivos innecesarios.
